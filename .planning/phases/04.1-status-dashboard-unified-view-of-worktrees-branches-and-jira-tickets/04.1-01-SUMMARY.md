## Decisions Made

- `gitListBranches` uses `--format=%(refname:short)` — produces clean branch names without the `* ` or `  ` prefix that `git branch` (without `--format`) adds, so no stripping is needed
- No `-C` flag in `gitListBranches` — runs in CWD following the same pattern as `gitBranchExistsOnRemote`
- `includeAll` defaults to `false` — ensures all existing callers (`pick.ts`) remain unaffected with no code changes required
