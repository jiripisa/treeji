# Quick Task 260406-g61: Worktree Add Existing Branch Summary

## What Was Built

`gitBranchExists(branch)` added to `src/lib/git.ts` using `git rev-parse --verify <branch>` — resolves `true` if branch exists locally, `false` on non-zero exit. `gitWorktreeAdd` updated to call `gitBranchExists` first, choosing the correct command form, and now returns `{ existed: boolean }`. Both `create` and `pick` commands destructure the return value and print `Using existing branch: <branch>` to stderr when `existed` is true.
